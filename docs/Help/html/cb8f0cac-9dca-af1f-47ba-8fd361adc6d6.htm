Remarks
You can register a service-related settings type that implements the ISettings interface.

The application is obliged to pass an instance of the specified type to the service constructor.

Examples
[Service(SettingsType = typeof(FooSettings))] public class Foo { private ISettings Settings { get; } public Foo(FooSettings settings) { Settings = settings; } }
